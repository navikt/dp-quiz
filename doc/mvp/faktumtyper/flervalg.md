# Flervalg

Typisk brukt for sjekkbokser (multiple choice).

#### JSON-eksempel med svar
```
    {
      "id": "7",
      "clazz": "flervalg",
      "navn": "flervalg7",
      "roller": [
        "søker"
      ],
      "svar": [
        "flervalg1",
        "flervalg2"
      ],
      "gyldigeValg": [
        "flervalg1",
        "flervalg2",
        "flervalg3"
      ]
    }
```

##### Feltforklaringer

##### gyldigeValg
Det er kun disse verdiene som er et gyldig svar, og det er tillatt å velge et eller flere alternativer.
